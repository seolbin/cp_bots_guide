블로그 봇 설정 가이드

몇 가지 간단한 설정을 완료하고, 편리한 자동 포스팅을 시작해보세요!

⚙️ 1. 필수 사전 준비

1

네이버 아이디 준비

봇이 블로그에 안전하게 로그인하기 위해 전용 아이디를 준비해야 합니다.

  • 2단계 인증 해제: 봇이 로그인할 수 있도록 2단계 인증이 해제되어야 합니다.
  • 새 아이디 사용 권장: 보안을 위해 봇 전용 아이디를 새로 만드는 것을 강력히 추천합니다.
2

쿠팡 파트너스 API 키 발급

모든 포스팅에 수익 링크를 포함하기 위해 쿠팡 파트너스 API 키가 필요합니다.

  1. 쿠팡 파트너스 로그인 후, [추가 기능 > 파트너스 API] 메뉴로 이동합니다.
  2. `Access Key`와 `Secret Key`를 생성하고, 두 키를 모두 복사하여 보관해주세요.
쿠팡 파트너스 바로가기 ➡️
3

Gemini API 키 발급

Google의 Gemini AI가 블로그 원고를 생성하기 위해 API 키가 필요합니다.

  1. Google AI Studio 사이트에 구글 계정으로 로그인합니다.
  2. "Get API Key" 또는 "Create API Key" 버튼을 클릭하여 새 키를 생성합니다.
  3. ⚠️ 중요: 과금 정보(Billing)를 등록하지 않고 무료 등급으로만 사용하세요.
Google AI Studio 바로가기 ➡️
